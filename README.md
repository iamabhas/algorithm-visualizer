## How to Run the Project

Follow these steps to run the project on your local machine:

## 1. Clone Project

```bash
git clone https://github.com/iamabhas/algorithm-visualizer.git
```
## 2. Install Dependencies

```bash
cd algorithm-visualizer
npm install
```
## 3. Run Project
```bash
npm run start
```
